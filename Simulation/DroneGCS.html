HOW TO GET THE .BIN FROM ARDUINO
Use the app firmware binary for OTA, usually named like:
RC_FlightController_OTA.ino.bin

Do not upload the bootloader, partitions, or merged factory image through OTA.
Arduino IDE steps
1. Select the correct ESP32 board. 2. Select an OTA-capable partition scheme, for example “Default 4MB with OTA”. 3. Build once with USB and flash the OTA-capable firmware. 4. For future updates: Sketch → Export Compiled Binary. 5. Sketch → Show Sketch Folder. 6. Pick the generated application .bin file. 7. Connect laptop to ESP32-DRONE Wi-Fi. 8. Upload the .bin from this tab.
