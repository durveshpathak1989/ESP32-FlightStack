How this screen is created

1. ESP32 prints one text telemetry line over USB serial.

2. Browser reads that line using Web Serial.

3. Parser extracts roll, pitch, yaw, RC commands, motors, mode, and RC Hz.

4. State object is updated.

5. Render function redraws meters, bars, sticks, motors, compass, and artificial horizon.

Use SIM MODE or sliders to test UI without ESP32.
